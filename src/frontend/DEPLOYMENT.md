# GB Insurance - Production Deployment Guide

This guide covers deploying the GB Insurance application to the Internet Computer mainnet.

## Prerequisites

1. **DFX CLI** installed and configured (version 0.15.0 or later)
2. **Cycles wallet** with sufficient cycles for deployment
3. **Admin token** for initial admin setup
4. **Node.js** (v18 or later) and **pnpm** installed

## Environment Configuration

### Backend Canister ID

The frontend needs to know the backend canister ID to connect properly. This is configured via:

1. **Development**: Automatically read from `env.json` (generated by dfx during `dfx generate`)
2. **Production**: Set via `VITE_BACKEND_CANISTER_ID` environment variable during build

### Network Configuration

- **Development**: `http://localhost:4943` (local replica)
- **Production**: `https://ic0.app` (IC mainnet)

The configuration is handled automatically in `frontend/src/config.ts` based on the build environment.
